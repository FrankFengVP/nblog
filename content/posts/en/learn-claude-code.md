I spotted a repo on GitHub Trending called **learn-claude-code**. The opening line stopped me cold:

> Agency — the ability to perceive, reason, and act — comes from model training, not from external code orchestration.

That line punctures a multi‑billion‑dollar illusion in the AI world over the past two years: products that wire an LLM API together with if‑else branches and node graphs and call it an “Agent.” The README is even sharper:

> What they build is a Rube Goldberg machine... That is not an agent. That is a shell script with delusions of grandeur.

## What the project is about

The project’s core pitch is **12 progressive sessions** that walk you through building a miniature **AI Agent engine** by hand. Five historical milestones support the claim that *intelligence comes from training, not coding*, and the conclusion is:

> Agency — the ability to perceive, reason, and act — is trained, not coded.

## Harness engineers: the model drives; you build the car

The setup leads to one point: as a **Harness engineer**, your job is not to program intelligence into the model, but to build the world the model can live in.

- The **model** is the driver
- The **Harness** is the vehicle

The project never breaks that rule: 12 sessions add 12 mechanisms, and **the core loop never changes by a single line**.
