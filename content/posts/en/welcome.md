Hello, and welcome.

NBlog is a lightweight blog built with **Next.js** and **Markdown**. Posts live under `content/posts/en` or `content/posts/zh`. Each file is a `.md` document with Front Matter for the title, date, and excerpt.

## How to write a post

Create a Markdown file under `content/posts/en`, for example `my-post.md`:

```markdown
---
title: "Post title"
date: "2026-05-19"
excerpt: "One-line summary"
tags: ["tag"]
---

Start writing here…
```

Save the file and refresh the page to see the new post.
